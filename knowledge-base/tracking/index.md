# Tracking

> Generated by `rake kb:index` — do not edit by hand; change the source file's `description` and regenerate.

* [Bugs & investigations](bugs.md) - Known live defects and things to investigate. These are correctness/performance fixes, distinct from dead schema, incomplete features, and refactors.
* [Dead schema](dead-schema.md) - Vestigial tables and columns that exist in the schema but are unused by live code and safe (eventually) to drop. Most trace to the deprecated Trinidad banking model.
* [Incomplete features](incomplete-features.md) - Schema for features that were started but never finished. The columns exist but drive no behavior. Not safe to drop (may be completed), not safe to rely on.
* [Open questions](open-questions.md) - Pending decisions and unresolved threads an agent should not silently resolve. Each names what's unknown and where it will be settled.
* [Refactor candidates](refactor-candidates.md) - Naming/structure improvements worth considering. Not defects — the system works as-is — but changes that would improve clarity, especially for agents.
